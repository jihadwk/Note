c# Lambda表达式 类似匿名函数
Lambda表达式可以有多个参数、一个参数，或者没有参数。其参数类型可以隐式或者显式。示例代码如下：

(x, y) => x * y         //多参数，隐式类型=> 表达式  
x => x * 5              //单参数， 隐式类型=>表达式  
x => { return x * 5; }      //单参数，隐式类型=>语句块  
(int x) => x * 5            //单参数，显式类型=>表达式  
(int x) => { return x * 5; }      //单参数，显式类型=>语句块  
() => Console.WriteLine()   //无参数
